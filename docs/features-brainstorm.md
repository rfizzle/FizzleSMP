# Feature Brainstorm — FizzleSMP

Use this document to capture feature ideas first, then map them to specific mods.

## How to Use

1. Add your idea under the relevant theme (or create a new theme).
2. Describe *what the player experience should be* — not which mod to use.
3. Once a mod is identified, link it here and add the full entry to `plugins/<category>.md`.
